# Bert_TextCNN_Classification

基于Bert和TextCNN的文本分类实现

## Dataset

ChnSentiCorp数据集

- 训练数据：9600条
- 验证数据：1200条
- 测试数据：1200条

标签：

- 正向：1
- 负向：0

数据的label和text之间通过text分割
